Development server

Clonar el projecto.

Ejecutar el script SQL situado en la carpeta php en el servidor y colocar los demás ficheros de dicha carpeta en un servidor apache. Modificar el fichero situado en src/app/services llamado urls.service.ts para indicar la url del servidor.

Acceder a la carpeta raíz desde el terminal del sistema. Ejecutar npm install para descargar los paquetes y dependencias necesarias. Ejecutar ng serve para iniciar la aplicación en modo desarrollo. acceder a http://localhost:4200/ desde cualquier navegador para ver la aplicación.
